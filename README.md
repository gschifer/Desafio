# Desafio-Back-end-IBM
Desafio Técnico Back-end para implementar uma API para votação de pautas entre os associados.

## Tecnologias utilizadas na aplicação

- **Spring Boot**
- **JPA**
- **Lombok**
- **Spring Security**
- **JUnit**
- **Bean Validation**
